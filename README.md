# RPGSys

A role-playing game management system designed to handle characters, attributes, inventory, combat mechanics, and other core RPG features.

## 📖 About the Project

RPGSys was developed to provide a structured and scalable foundation for RPG-based applications and games. The project centralizes character management and gameplay mechanics, making it easier to build and maintain RPG systems.

## ✨ Features

- Character creation and management
- Attribute and statistics system
- Experience and leveling mechanics
- Inventory management
- Combat system
- Data persistence
- Modular architecture for future expansion
